ELITES 課題005「文字列からN番目」
==================================

## 課題内容
### 基本要件(必須)
長さm の半角アルファベット文字列s と 正の整数n が入力された時、  
文字列s の1番左の文字を1文字目として、n文字目のアルファベットを出力せよ。  
ただし、n が m よりも大きい値をとる場合は、"m以下 正の整数を入力してください"と出力せよ。  

[実行結果例]  
s = "hoge", n = 3 g  
s = "hoge", n = 5 4以下の正の整数を入力してください
